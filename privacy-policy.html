Privacy Policy

URL Traversal Timer Chrome Extension

URL Traversal Timer does not sell or share user data with third parties. This extension is designed to run locally in your browser and use only the data needed to cycle through a user-defined list of URLs at a chosen interval.

Information We Collect

The extension may store the following information locally on your device:

  • URLs you enter into the extension
  • The active tab URL used to initialize the extension
  • Traversal settings such as interval selection and running state
  • Website content in the form of user-entered URLs and the currently active tab URL

How We Use This Information

We use this information only to provide the extension’s core function:

  • Save your URL list and settings
  • Rotate the active tab through your selected URLs
  • Pause, resume, and stop traversal correctly
  • Remember your configuration between sessions

Where the Data Is Stored

All stored data is kept locally in Chrome’s extension storage on your device. It is not uploaded to any developer server because this extension does not use a backend service.

Data Sharing

We do not sell, transfer, or disclose user data to third parties, except where required by law.

Permissions

The extension uses Chrome permissions only for its stated purpose:

  • storage to save the local URL list and settings
  • tabs to read the active tab URL and switch to the next URL
  • alarms to keep timing reliable in the background

Children’s Privacy

This extension is not intended for children under 13, and we do not knowingly collect personal information from children.

Changes to This Policy

If this privacy policy changes, we will update this page with the revised version.
